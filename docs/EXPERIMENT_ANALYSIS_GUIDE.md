# 실험 결과 분석 가이드

## 📊 실험 결과 도출 및 비교 방법

### 1. 최종 데이터 도출

**주요 평가 지표**:

- **ADE (Average Displacement Error)**: 평균 위치 오차 (m)
- **FDE (Final Displacement Error)**: 최종 위치 오차 (m)
- **Miss Rate**: 목표 지점 도달 실패율 (%)
- **Collision Rate**: 충돌 예측율 (%)
- **Safety Metrics**: TTC, PET, DRAC (Plan B)

**다중 모달리티 평가**:

- **Diversity**: 생성된 궤적의 다양성 (평균 쌍별 거리)
- **Coverage**: 실제 궤적 커버리지 (K=20 샘플 기준)

### 2. 비교 대상 (Baseline)

**베이스라인 모델**:

1. **Social-STGCNN**: 기본 GNN 기반 모델
2. **Trajectron++**: CVAE 기반 다중 모달리티 모델
3. **A3TGCN**: 현재 프로젝트의 GNN 모델 (단일 모달리티)
4. **MID**: 원본 MID 모델 (동종 에이전트)
5. **우리 모델**: GNN-MID 하이브리드 (이기종 환경)

**데이터셋별 비교**:

- SDD Death Circle: 이기종 에이전트 환경
- 에이전트 타입별 성능 (차량, 보행자, 자전거 등)

### 3. 결론 도출 방법

**통계적 유의성 검증**:

- 각 지표별 평균 ± 표준편차
- t-test 또는 Wilcoxon test로 유의성 검증 (p < 0.05)
- 시나리오별 성능 분석 (회전교차로 특화 상황)

**최종 결론**:

1. **성능 향상**: 우리 모델이 베이스라인 대비 ADE/FDE 개선 정도
2. **다중 모달리티**: MID 통합으로 다양성/커버리지 향상 여부
3. **이기종 처리**: 이기종 환경에서의 성능 우위
4. **안전성**: Plan B 통합으로 위험 궤적 감소율

## 📈 간단 요약 (3문장)

**1. 최종 데이터 도출**: ADE, FDE, Miss Rate, Collision Rate, Diversity 등 주요 지표를 계산하고, 에이전트 타입별 및 시나리오별로 성능을 분석합니다.

**2. 비교 대상**: Social-STGCNN, Trajectron++, A3TGCN, 원본 MID 등 베이스라인 모델과 비교하며, 특히 이기종 환경에서의 성능 차이를 분석합니다.

**3. 결론 도출**: 통계적 유의성 검증(p < 0.05)을 통해 우리 모델(GNN-MID 하이브리드)이 베이스라인 대비 성능 향상, 다중 모달리티 생성 능력, 이기종 환경 적응성, 안전성 향상을 입증합니다.
