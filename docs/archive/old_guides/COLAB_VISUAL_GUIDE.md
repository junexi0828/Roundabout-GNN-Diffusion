## 단계별 상세 프로세스

### Phase 1: 환경 설정 (5분)
```
셀 1 실행
  ├─ pip install (라이브러리)
  ├─ GPU 확인
  └─ ✓ 완료

셀 2 실행
  ├─ git clone 또는 업로드
  └─ ✓ 완료

셀 3 실행
  ├─ Google Drive 마운트
  └─ ✓ 완료
```

### Phase 2-3: 데이터 준비 및 전처리 (40분)
```
셀 4 실행
  ├─ 데이터 파일 확인
  └─ ✓ 완료

셀 5 실행
  ├─ 데이터 통합
  ├─ 전처리 실행
  ├─ 윈도우 생성
  └─ ✓ 완료 (저장됨)
```

### Phase 4: 모델 학습 (1-4시간) ⚡
```
셀 6 실행
  ├─ 데이터 분할
  ├─ DataLoader 생성
  └─ ✓ 완료

셀 7 실행
  ├─ 모델 생성
  ├─ Trainer 설정
  ├─ 학습 시작
  │   ├─ Epoch 1/50...
  │   ├─ Epoch 2/50...
  │   └─ ... (진행 중)
  └─ ✓ 완료 (체크포인트 저장)

셀 8 실행 (별도 탭)
  └─ TensorBoard 모니터링
```

### Phase 5-6: 평가 및 다운로드 (35분)
```
셀 9 실행
  ├─ 모델 로드

셀 10 실행
  ├─ 결과 다운로드
  └─ ✓ 완료
```
